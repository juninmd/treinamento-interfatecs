# Treinamento-Interfatecs

**A standard software project.**

**Installation:**

1.  Clone the repository: `git clone https://github.com/interfatecs/treinamento-interfatecs.git`
2.  Navigate to the project directory: `cd treinamento-interfatecs`
3.  Install dependencies (if any): `pip install -r requirements.txt`

**Usage:**

*   **Configuration:**  Modify the configuration file `config.json` to customize the project's behavior.  Key settings include:
    *   `model`: Specifies the model to be used (e.g., "YOLOv5").
    *   `output_dir`: Sets the directory for the generated output.
*   **Run:** Execute the script: `python main.py`
*   **Testing:**  Verify the functionality through the included tests.  You can view the tests in the `tests` directory.
